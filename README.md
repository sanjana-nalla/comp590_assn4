## Description
This Erlang program creates a sequence of three servers: **serv1**, **serv2**, and **serv3**.

### Server Responsibilities

- **serv1**  
  Performs arithmetic calculations.

- **serv2**  
  Computes the sum and product of elements in a list.

- **serv3**  
  Manages errors and keeps track of unprocessed messages.
